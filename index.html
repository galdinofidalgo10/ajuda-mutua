Estatuto e Funcionamento do Projeto
1. O Gatilho de Ativação Único

Para entrar no sistema e validar sua posição, o membro realiza 3 doações diretas de R$ 5,00 para os líderes da linha ascendente. Isso totaliza um custo único de R$ 15,00. O ID de divulgação só é liberado após o upload dos comprovantes e a auditoria manual de 24 horas.

2. Estrutura da Matriz Forçada (3x10)

O sistema organiza os participantes em uma matriz de 10 níveis. Você pode ter apenas 3 pessoas diretamente ligadas ao seu primeiro nível. Todo o crescimento além disso beneficia a rede através do preenchimento automático de vagas.

3. O Poder do Derramamento (Transbordo)

Como cada membro só pode ter 3 diretos, novos convidados "derramam" automaticamente para as vagas vazias dos níveis abaixo. Isso ajuda quem tem mais dificuldade em convidar a começar a receber doações.

4. Fluxo de Dinheiro Direto (P2P via Pix)

O sistema não retém dinheiro. As doações são feitas no modelo Pessoa para Pessoa (P2P), enviadas diretamente para a conta Pix do recebedor. Isso garante liquidez imediata.

5. Auditoria de Segurança e Banimento

O sistema rastreia o IP e o CPF de cada usuário. Existe tolerância zero para fraudes: o envio de comprovantes falsos resultará no bloqueio imediato do cadastro.

6. Escritório Virtual e Sustentabilidade

Ao ser ativado, o membro ganha acesso ao seu painel para monitorar a rede e acompanhar as ativações descendentes. O projeto é sustentável porque organiza o fluxo de doações voluntárias de forma inteligente.
